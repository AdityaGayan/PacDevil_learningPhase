# PacDevil_learningPhase
## Run the project

To view the final game, clone this repository and run the following commands in terminal
```bash
cd PacDevil_learningPhase/task1
```
Then
```bash
npm run dev
```
And then go to http://localhost:3000/

Note:
1) Character is at the bottom left of the game screen(at first it might be difficult to view because color of ground and character is same)
2) the main typescript file game.ts is in src directory
